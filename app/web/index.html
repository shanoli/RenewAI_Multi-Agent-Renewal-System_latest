Escalation — Human Queue

❓ How Escalation Works
🚨
Why triggered?
• distress_flag = true
• objection_count ≥ 3
• HNI grievance detected
• Critique A override ×2
• Critique B → ESCALATE verdict
• 3+ channel attempts, no reply
⚙️
What happens?
• AI workflow pauses
• mode = HUMAN_CONTROL
• Case created in DB
• Priority score assigned
• Briefing note generated
• Specialist notified
🕐
SLA & Priority
• Priority ≥ 0.9 → 2h SLA
• Priority 0.7–0.9 → 4h SLA
• Priority < 0.7 → 8h SLA

• Score: distress=1.0
• HNI grievance=0.9
• Critique override=0.7
✅
After Escalation
• Human resolves + notes
• mode reset → AI
• distress_flag cleared
• policy_state updated
• AI resumes at Orchestrator
• Audit trail updated
